{% load inventree_extras %} {% comment %} Base de TOUS les e-mails HTML natifs d'InvenTree, surchargee par le plugin « E-mails » (chargeur de templates de plugin, prioritaire sur le template natif du meme chemin email/email.html). Restyler cette base restyle d'un coup tous ses enfants (stock bas, peremption, evenement piece, commandes...). DA reprise du plugin Prets (prets/email/notification.html) : page grise, carte blanche a lisere d'accent, pastille d'etat, pied « Une solution EirSpace ». HTML d'e-mail = tables + styles inline uniquement (aucune feuille de style, aucune image distante : bloquees par les clients mail). Compatibilite : on garde TOUS les noms de blocs de la base native d'InvenTree (header, header_row, title, subtitle, body, body_row, footer, footer_prefix, footer_suffix) pour que les enfants non surcharges continuent de s'afficher. La carte est une a fond blanc : les injectes par les enfants (ex. lignes de stock bas) reposent donc sur le blanc, sans retouche. Blocs de couleur surchargeables par enfant : card_accent (lisere gauche), pill_fg / pill_bg (pastille), section (surtitre a cote du nom d'instance), pill (libelle de la pastille). Le bandeau affiche le NOM DE L'INSTANCE (reglage InvenTree INVENTREE_INSTANCE, pose par create-asso.sh au nom de l'asso). S'il vaut encore le defaut, l'e-mail affiche donc « InvenTree ». Textes en francais litteral : voir la section « Langue » du README. {% endcomment %}
